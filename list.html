*bydefault list tag is indicated with dot

*list has two types

  • 1.ordered list
  • 2.unordered list

    • *ordered list is indicated with ol tag

    *unordered list is indicated with ul tag

    *bydefault for ordered list it is indicated by numbers

    *bydefault for unordered list it is indicated by dots

    *if we need any particular symbols or constants we use type="symbol or constants"(alphabets,numbers)

    *if we doesnt give list tag for every element then it will print like side by side (HTML CSS JAVASCRIPT JAVA MYSQL)without numbers and constants

    *In ordered list we have to start with any ordered pairs(like numbers and alphabets)

    *if we want to start with middle of the constants we have to include start="number"
